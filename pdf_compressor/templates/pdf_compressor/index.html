PDF Compression API

This API provides PDF compression functionality. Send your PDF files to be compressed and receive an optimized version in return.

Endpoint

/api/compress/

Method

POST

Request Format

Send a multipart/form-data request with the following parameter:

  • pdf_file: The PDF file to be compressed

Example Request

curl -X POST \
  -F "pdf_file=@/path/to/your/file.pdf" \
  http://localhost:8000/api/compress/

Response

On success, receives the compressed PDF file with Content-Type: application/pdf

Error Responses

  • 400 Bad Request: Invalid file type or missing file
  • 500 Internal Server Error: Error during compression

File Size Limits

Maximum file size: 10MB
